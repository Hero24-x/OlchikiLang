## 📌 Overview

**OlchikiLang** is an open-source programming language engineered to bridge the gap between computer science and indigenous scripts. By enabling software development directly within the **Ol Chiki** writing system, it provides Santali speakers with a native environment to learn, build, and innovate using localized keywords, numerals, and syntax grammar.

### Key Goals
* **Democratize Coding:** Remove language barriers for Santali native speakers entering the tech space.
* **Native Integration:** Full Unicode support for Ol Chiki script and numerical systems.
* **Ecosystem Growth:** Moving from a lightweight scripting runtime to a full-stack development ecosystem.

---

## 🚀 Core Features

* **Complete Control Flow:** Supports conditional branching (`ᱡᱩᱫᱤ`/`ᱮᱞᱥᱮ`) and iterative loops (`ᱫᱚᱦᱚᱨ`).
* **First-Class Functions:** Lexical scoping, parameters, nested calls, and explicit returns.
* **Unicode & Numerals:** Built-in lexer support for native Ol Chiki numbers (`᱐-᱙`).
* **Developer Tooling:** Includes an interactive REPL environment for rapid prototyping and testing.
