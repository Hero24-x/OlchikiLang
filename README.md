## About

OlchikiLang is a programming language created to enable software development using the Ol Chiki writing system.

The goal of the project is to provide Santali speakers with a native programming environment where coding can be performed using Santali keywords, Santali numerals, and Ol Chiki script.

# Example Programs

## Print

```olchiki
ᱪᱟᱯᱟ "ᱡᱚᱦᱟᱨ"
```

## Variables

```olchiki
ᱥᱮᱴ ᱠ = ᱑᱐
ᱪᱟᱯᱟ ᱠ
```

## Arithmetic

```olchiki
ᱪᱟᱯᱟ (᱒ + ᱓)
```

## If Statement

```olchiki
ᱡᱩᱫᱤ (᱑ == ᱑) {
    ᱪᱟᱯᱟ "ᱦᱚᱴᱟ"
}
```

## If Else

```olchiki
ᱡᱩᱫᱤ (᱑ == ᱒) {
    ᱪᱟᱯᱟ "ᱦᱚᱴᱟ"
}
ᱮᱞᱥᱮ {
    ᱪᱟᱯᱟ "ᱵᱟᱝ"
}
```

## Loop

```olchiki
ᱥᱮᱴ ᱠ = ᱐

ᱫᱚᱦᱚᱨ (ᱠ < ᱓) {

    ᱪᱟᱯᱟ ᱠ

    ᱥᱮᱴ ᱠ = ᱠ + ᱑
}
```

## Functions

```olchiki
ᱯᱩᱱ ᱡᱚᱜ(ᱠ, ᱜ) {

    ᱫᱟᱨ ᱠ + ᱜ
}

ᱪᱟᱯᱟ ᱡᱚᱜ(᱒, ᱓)
```
